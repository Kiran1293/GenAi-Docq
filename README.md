GenAi-Docq is an intelligent document question-answering system powered by Generative AI. It allows users to upload documents and query them using natural language for fast, relevant answers.

---

## 🚀 Features

- Upload PDF/DOCX for contextual Q&A
- Ask natural language questions
- Answer extraction with confidence scoring
- Streamlit or web-based frontend

---

## 🛠 Tech Stack

- 🐍 Python 3.x  
- 🤗 HuggingFace Transformers  
- 🧠 OpenAI / LangChain (optional)  
- 🖼️ Streamlit / Flask  
- 📄 PyMuPDF / Docx2txt  
- 🔍 FAISS (if vector search used)
